Calculating portfolio returns

Introduction

Calculating the return of a portfolio can be rather easy, as long as there are no cash flows involved. In this case, we only need the following input: initial value of the investment $V_0$, current value of the investment $V_t$ and a year fraction $\tau$. The year fraction $\tau$ will express how much time (in years) has passed between initial valuation at time $0$ and the current valuation at time $t$. The return $r$ is then, $$ r = \frac{1}{\tau}\left(\frac{V_t}{V_o} - 1\right). $$
